Linux Commands Recreated in C

This demo page explains my recreation of some essential Linux commands in C, showcasing the functions used and a summary of each command's purpose.

my_ls

Reimplementation of the ls command to list directory contents.

Used functions:

  • opendir
  • readdir
  • stat
  • closedir
  • my_getopt

my_top

Recreation of top to display running processes in the terminal.

Used functions:

  • opendir
  • readdir
  • closedir
  • mvprintw
  • get_pwuid

my_sudo

Simple version of sudo implemented collaboratively with a peer.

Used functions:

  • getuid
  • getgid
  • fopen / fclose
  • strtok

Note: This project was completed as a two-person assignment.
