About Me

I am an IT student in my 2nd year at Epitech La Réunion, a prestigious programming school where I've been developing strong technical skills and hands-on project experience.

Throughout my studies, I've completed several challenging C projects, including recreating essential Linux commands such as ls, top, and sudo, as well as recoding the fundamental printf function. I also developed a graphical project by recreating the classic game Duck Hunt using CSFML.

As a TekTek ambassador, I actively represent the school during open days, forums, and school presentations, helping prospective students discover the opportunities at Epitech.

I completed an internship at the DSI of a city administration, where I gained valuable web development experience by creating an employee directory with advanced features including card layouts, sorting, authentication, and search functionality using HTML, CSS, JavaScript, and Python.

I've also worked on mobile development, creating a mobile/tablet application in Kotlin on Android Studio using collaborative workflows with GitHub and Git. This experience strengthened my teamwork and version control skills.

Beyond technical development, I've improved my communication skills by helping users, answering calls, and handling support requests.
